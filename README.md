# 💡 Light Intensity Display on LCD using Arduino

## 📌 Description
This project measures **light intensity** using an LDR (Light Dependent Resistor) and displays the values on a **16x2 LCD display** using Arduino. The system continuously monitors the surrounding light and updates the intensity level in real time.

---

## 🔧 Features
- 💡 Detects light intensity using LDR  
- 📟 Displays values on 16x2 LCD  
- ⚡ Real-time monitoring  
- 🔄 Continuous data update  
- 📊 Easy to understand output  

---

## 🧰 Components Used
- Arduino Uno  
- LDR (Light Sensor)  
- 16x2 LCD Display  
- Potentiometer (for contrast)  
- Resistor (10kΩ recommended)  
- Breadboard  
- Jumper Wires  

---

## ⚙️ Working Principle
1. LDR senses the light intensity  
2. Resistance changes based on light level  
3. Arduino reads analog values from LDR  
4. Values are processed and converted  
5. LCD displays the light intensity
